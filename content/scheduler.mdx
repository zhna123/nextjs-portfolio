This project incorporates new features from Next.js and React, including the App Router and Server Actions. 
It enables users to choose a schedule, enter the time through a pop-up form, and securely submit the data 
without the need for separate API endpoints. The data is written and configured using a JSON file.

## Key Features

- Users can select a schedule and enter a time via a pop-up form.
- Data is written and stored in a JSON file for easy configuration.
- The popup modal is implemented using search params to leverage server-side rendering (SSR).
- Server Actions handle form submission, ensuring secure data handling without the need for API endpoints.

## Technologies Used

- Next.js (App Router, Server Actions)
- React
- TypeScript
- TailwindCSS
- JSON for storing scheduling data
