This project is *in progress*.  
Currently, work is focused mostly on:

- Enhancing the user interface for lunchbox and user management
- Integrating with external nutrition APIs
- User authentication

## Key Features

- User and lunchbox management
- Food/recipe management with personalized food storage
- External API integration for common food and nutrition info
- Image storage via Amazon S3
- Secure user authentication with Spring Security

## Technologies Used

### Backend:
- Java 22
- Spring Boot 3.3
- Spring Data JPA
- Spring Security
- Amazon S3
- Maven
- JUnit

### Frontend:
- React
- CSS Modules
- TanStack Query
