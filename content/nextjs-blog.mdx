- Used Next.js page router and Static Site Generation.
- Styled with TailwindCSS.
- The content is written with MDX, and is loaded using *next-mdx-remote*.
- Implemented pagination with SSG.
- Contact Form is implemented using API Routes and the data is stored in Vercel Postgres.
