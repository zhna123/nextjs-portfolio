Orbit Talk is a messaging web app that enables users to send messages to each other. 
The frontend is built with React and TypeScript, while the backend is powered by 
a RESTful API implemented using Express and MongoDB. The app uses JWT for secure user authentication 
and focuses on web accessibility throughout the development process.

## Key Features

- Users can send messages to their friends in a chat interface.
- User authentication is handled with JWT for secure login and session management.
- Backend implemented with Express and MongoDB, providing a RESTful API.
- Frontend developed with React and TypeScript for a robust user experience.
- Accessibility was a key focus, ensuring the app is usable for a wide range of users.
- API endpoint unit testing with Jest and Supertest.
- Frontend testing with Vitest and React Testing Library.

## Technologies Used

### Frontend:
- React
- TypeScript
- Vitest
- React Testing Library

### Backend:
- Express
- MongoDB
- JWT (JSON Web Token)

### Testing:
- Jest
- Supertest
