- Used React and Typescript for frontend deveopment.
- Backend RESTful API was implemented using Express and MongoDB.
- Used JWT for user authentication.
- Router testing with Jest and supertest for the API endpoints unit testing.
- Vitest and React Testing Library for frontend testing.
- Maintain a focus on accessibility for the app and learned more about web accessibility.
